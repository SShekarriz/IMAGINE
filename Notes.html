Inset Plots

I found a stack overflow question about this.

To do inset graphs (e.g. superimpose the bar graph over the scatterplot), we use `grid::viewport()`` apparently. Link here.

Another (better?) option is to use the egg library:

ps_rel <- ps %>%
transform_sample_counts(function(x) {x/sum(x)} )# Transform to rel. abundance

bray_dis_ps = phyloseq::distance(ps_rel, method = 'bray')
bray_ord_ps = ordinate(ps_rel, method = 'PCoA', distance = bray_dis_ps)


plt = plot_ordination(ps_rel, bray_ord_ps, color="Status") +
    scale_color_manual(values = c("Healthy" = "#bdbdbd",
                                  "IMG106" = "#1a9850")) +
    geom_point(alpha = 0.5, size=3) +
    ggplot2::theme_classic() +
    theme(legend.title = element_blank(),
          legend.text  = element_text(size=15),
        text = element_text(size = 10),
        legend.position = "right")
